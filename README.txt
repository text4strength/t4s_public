Text4Strength Source Code 
=========================
This Github repo contains the entirety (minus credentials) of the software/code for Text4Strength (T4S) web application.


T4S was developed in 2012-2014 and was used in academic research to distribute multi-level text message sequences to study participants.


To run it properly, user must install Python 2 and its associated packages detailed in 'requirements.txt'.


The software is distributed under GNU GPL v3.0.
